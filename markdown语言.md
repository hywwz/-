
## 基础语法

### 1. 标题

```markdown
# 一级标题   后面要带一个空格，否则就是小标签
## 二级标题
### 三级标题
```

### 2. 文字样式

```markdown
**粗体文本** 或 __粗体文本__ ctrl b
*斜体文本* 或 _斜体文本_ ctrl i
***粗斜体*** 或 ___粗斜体___
~~删除线文本~~
==高亮文本== (部分编辑器支持)
`DSKFJJHASDKLFD` 一个`就是代码条，三个就是代码块，在后面标明语言
```

### 3. 列表

#### 无序列表
```markdown
- 项目一 记得有空格
- 项目二
  - 子项目一
  - 子项目二
```

#### 有序列表
```markdown
1. 第一项 记得加空格
2. 第二项
   3. 子项一
   4. 子项二
5. 第三项
```

### 4. 链接和图片

```markdown
[带标题的链接](https://example.com "标题") ctrl+k 在阅读模式中打开

直接拖拽或者复制粘贴
```

### 5. 引用块

```markdown
> 这是一个引用块 记得加空格
> 可以多行使用
>
> > 嵌套引用
> 
> - 引用中的列表
> - 另一个项目

```

### 6. 任务列表

```markdown
- [x] 已完成任务 不要有空格就行
- [ ] 未完成任务
- [ ] 另一个任务
```

### 7. 脚注

```markdown
这是一个带有脚注的句子[^1]。

[^1]: 这是脚注的内容。
```
