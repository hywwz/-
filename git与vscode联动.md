### 一、 安装Git

在你做任何事情之前，需要先安装Git：

1. 访问?[https://git-scm.com/](https://git-scm.com/)
    
2. 点击下载，然后一路"Next"安装即可
    
3. 安装完成后，在VSCode中按?`` Ctrl+` ``?打开终端，输入：
    
    bash
    
    git --version
    
    如果显示版本号（如?`git version 2.xxx`），说明安装成功。
    
### 二、GitHub仓库创建

1. 登录GitHub，点击右上角?**+**?->?**New repository**
    
2. **Repository name**: 取一个清晰的项目名，如?`my-todo-app`,?`algorithm-practice`
    
3. **Description**: 简短描述项目是什么
    
4. **Public/Private**: 选择?**Public**（让全世界看到你的作品！）
    
5. **勾选**?"Add a README file"（最重要的一步！）
    
6. **勾选**?"Add .gitignore"，选择你的语言模板（如Python, Java）
    
7. **勾选**?"Choose a license"，初学者可选MIT License
    
8. 点击?**Create repository**

### 三、克隆

假设你已经按照之前的指导，在GitHub上创建了一个名为?`my-first-project`?的仓库。
建议创建一个专门的文件夹，比如?`D:\MyProjects\`?或?`~/Documents/code/`

1. **获取仓库URL**：
    
    - 打开你的GitHub仓库页面
        
    - 点击绿色的?**"Code"**?按钮
        
    - 复制那个以?`https://github.com/...`?开头的URL  


2. **在VSCode中操作**：
    
    - 按?`Ctrl+Shift+P`?打开命令面板
        
    - 输入?`Git: Clone`?然后按回车
        
    - 这时会弹出一个小窗口，让你粘贴URL
        
    - 把刚才复制的URL粘贴进去，按
        
3. **选择存储位置**：
    
    - 接下来会让你选择把库克隆到电脑的哪个位置
        
    - 点击刚才创建的专门的文件夹
        
    - 选择好文件夹后点击"确定"
        
4. **打开项目**：
    
    - 克隆完成后，VSCode会问你是否要打开这个项目
        
    - 点击?**"Open"**?即可
        

### 四、提交至github

### “暂存 -> 提交 -> 推送” 完整流程

把这个过程想象成寄快递：
- **暂存** = 把要寄的物品打包
- **提交** = 贴上快递单（写清楚里面是什么）
- **推送** = 真正把包裹寄出去

---

#### 第一步：暂存 - 选择哪些文件要上传

**在VSCode中的操作：**

1.  **修改或创建文件**：比如你创建了 `test.py` 或修改了 `README.md`
2.  **打开源代码管理面板**：按 `Ctrl+Shift+G`
3.  **查看"更改"**：你会看到所有被修改的文件列表
4.  **暂存文件**：
    - 在文件名旁边点击 **"+"** 号（暂存单个文件）
    - 或者点击"更改"上方的 **"全部暂存"**（暂存所有文件）

**你会看到**：文件从"更改"区域移动到了"暂存的更改"区域。

---

#### 第二步：提交 - 给这次修改写备注

**在VSCode中的操作：**

1.  **在上方输入框写提交信息**：
    - 要清晰描述这次修改，比如：
      - `"添加用户登录功能"`
      - `"修复首页布局bug"`
      - `"更新项目文档"`
    - **不要写**模糊的信息如`"更新"`、`"修复"`

2.  **点击勾号提交**：点击输入框上方的 **√** 图标（或者按 `Ctrl+Enter`）

**完成效果**：此时更改已经保存在你的**本地Git仓库**中，但还没有上传到GitHub。

---

### 第三步：推送 - 上传到GitHub

**在VSCode中的操作：**

1.  **点击推送按钮**：循环符号
2.  **等待完成**：底部状态栏会显示推送进度

Git还・需要知道是谁提交的代码。

1. 在VSCode中按?`` Ctrl+` ``?打开终端
    
2. 依次输入以下两个命令（替换成你的信息）：
    ` git config --global user.name "你的GitHub用户名"
    ` git config --global user.email "你的GitHub注册邮箱"

##### 你以后的操作步骤：

1. **打开VSCode**
    
2. **直接打开你的项目文件夹**（从最近文件列表，或者?`文件 -> 打开文件夹`）
    
3. **开始编码**
    
4. 完成后：暂存 -> 提交 -> 推送
